# Storyboard: Stop Loss Explained

Category: Investor Toolkit
Brand Tagline: Build Wealth. Not Debt.
Audience: Young Investors (20-35)
Logo: assets/logos/logo.png

Scene 1:
Hook: One wrong order click can change your risk before you even notice it.

Scene 2:
Problem: A young investor sees a fast-moving price, enters quickly, and only later asks, "How much could I lose here?"

Scene 3:
Explanation: Stop Loss is a feature that helps you control how an order behaves, but it still needs a clear plan and risk limit.

Scene 4:
Example: If you are placing an order worth ₹10,000, decide the entry, exit, and maximum loss before using the feature. The tool should follow your plan, not replace it.

Scene 5:
Takeaway: Use stop Loss as a decision filter, not as a shortcut.

Scene 6:
CTA + Logo: Share this with someone learning order types. Use assets/logos/logo.png exactly as supplied. Add: Educational content only. This is not financial advice.
