# Layout Alignment Rules

## Production Alignment

Production mode uses consistent left alignment.

- Text starts at `x=120`
- Key message begins in the upper-middle region
- Supporting message follows with a 120px visual gap
- Key line spacing is 112px
- Support line spacing is 68px
- Outer content panel remains inside mobile-safe margins

## Visual Balance

- Key message: maximum two lines
- Supporting message: maximum two lines
- No category label in production scenes
- No scene number in production scenes
- No unnecessary footer content on Scenes 1-5
- CTA, tagline, disclaimer, and official logo use a consistent Scene 6 placement

## Logo

The only allowed logo is `assets/logos/logo.png`.

It is rendered with preserved aspect ratio and is never cropped, recolored, recreated, or replaced.
