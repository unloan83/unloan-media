# Text Hierarchy Rules

Every production scene uses exactly two editorial layers.

## Key Message

- Large, bold, and visually dominant
- Preferred length: 3-6 words
- Maximum length: 8 words
- Maximum: 2 lines
- Preferred size: 94px
- Minimum size: 78px

The key message carries the single idea viewers should remember.

## Supporting Message

- Regular or medium weight
- Preferred length: 6-12 words
- Maximum length: 14 words
- Maximum: 2 lines
- Preferred size: 42px
- Minimum size: 34px

The supporting message explains the key phrase without repeating it.

## Automatic Formatting

The hierarchy engine:

1. Normalizes currency and financial acronyms.
2. Detects the primary clause or question.
3. Extracts a concise key phrase.
4. Moves remaining context into support text.
5. Compacts common filler phrases.
6. Enforces the 8/14/20-word limits.

Expert-reviewed message sets are applied to the initial pilot topics. Other topics use the automatic hierarchy extractor.
