# UNLOAN Video Design System

## Brand Foundation

- Logo: `assets/logos/logo.png`
- Tagline: Build Wealth. Not Debt.
- Audience: Young Investors (20-35)
- Language: English
- Format: 1080x1920 vertical

The original logo must not be recolored, redesigned, recreated, cropped, substituted, or replaced.

## Palette

| Token | Value | Use |
| --- | --- | --- |
| Background primary | `#07111F` | Outer canvas |
| Background secondary | `#0F172A` | Main content panel |
| Text primary | `#FFFFFF` | Headlines and body copy |
| Text accent | `#D4AF37` | Category, rule, emphasis |
| Positive | `#65C466` | Positive financial states |
| Negative | `#E14B4B` | Risk or loss only |
| Border accent | `#D4AF37` | Subtle panel border |

## Typography

- Key message: Montserrat Bold, preferred 94px, minimum 78px
- Supporting message: Poppins Medium, preferred 42px, minimum 34px
- Supporting text: Poppins, preferred 34px, minimum 30px
- CTA: Montserrat Bold, preferred 48px, minimum 40px
- Disclaimer: Poppins, preferred 22px, minimum 20px

The engine does not shrink text below these minimums to force content into a scene.

## Layout

- Safe horizontal margin: 90px minimum
- Safe top and bottom margin: 120px minimum
- One key message
- One supporting statement
- One focal panel
- Official logo in its original proportions on the final scene
- No visible scene numbers in production mode
- No decorative clutter or excessive labels

## Modes

Production mode is the publishing preset. It uses clean layouts, simplified text, and no diagnostics.

Debug mode may show scene numbers and safe boundaries for internal inspection.
