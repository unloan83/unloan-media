# Readability Limits

## Per-Scene Limits

| Layer | Preferred | Maximum |
| --- | --- | --- |
| Key message | 3-6 words | 8 words |
| Supporting message | 6-12 words | 14 words |
| Total scene text | 10-16 words | 20 words |

Paragraph-style scene copy is not permitted.

## Typography Gates

- Key message: minimum 78px
- Supporting message: minimum 34px
- CTA: minimum 40px
- Disclaimer: minimum 20px

The key font must be at least 1.75 times the supporting font size.

## Failure Conditions

Production rendering fails when:

- Any word maximum is exceeded
- Either text layer exceeds two lines
- More than two primary text blocks are present
- Typography falls below minimums
- Key text is not visually dominant
- Alignment differs from the production preset
- Text enters configured safe margins
- Scene numbers appear in production mode
